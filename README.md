# react-memo-app

## 概要

以下の機能を備えたシンプルなメモアプリです。

- 一覧
- 詳細
- 追加
- 編集
- 削除

## 使い方

### 一覧

追加済みのメモがあれば、メモの１行目をタイトルとして一覧表示します。

### 詳細

一覧画面からタイトルをクリックすると、そのメモの詳細画面に移行します。

### 追加

＋ボタンをクリックすると、「新規メモ」というメモが作成され、そのメモを編集できます。

### 編集

テキストエリアの内容を編集し、編集ボタンをクリックすると保存されます。

### 削除

編集状態から削除ボタンをクリックすると、そのメモが削除されます。
